安装

Houttuyniae

# 安装

安装 Docker 要求 CentOS 系统的内核版本在 3.10 以上(一般在 CentOS 7 以上版本)

# Linux 下安装

# 查看内核版本

通过 uname -r 命令查看内核版本，版本高于等于我的即可

# 更新 yum 包

运行 yum -y update 命令 ，使用 root 权限登录 Centos。确保 yum 包更新到最新，否则会出现很多未知问题。

# 卸载旧版本

如果之前已经安装过 Docker,可以使用此命令删除

yum remove docker docker-common docker-selinux docker-engine

# 安装需要的软件包

yum install -y yum-utils device-mapper-persistent-data lvm2

# 设置 yum 源

yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

# 查看所有仓库中所有 docker 版本

yum list docker-ce --showduplicates | sort -r

# 安装

由于 repo 中默认只开启 stable 仓库，故这里安装的是最新稳定版 19.03.1

sudo yum install -y docker-ce

# 查看版本

docker version

# 启动并加入开机启动

systemctl start docker systemctl enable docker

# 镜像

如果使用官方的镜像地址，下载速度会特别慢，这里我们配置阿里镜像

# 配置阿里镜像

详细可以参考 https://cr.console.aliyun.com/cn-hangzhou/mirrors

sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://6uxh0lql.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker

# windows 下安装

# 说明

在 Windows 10 家庭中文版 下安装 docker

# 开启 Hyper-V

新建hyperv.cmd文件，内容如下：

pushd "%~dp0"

dir /b %SystemRoot%\servicing\Packages\*Hyper-V*.mum >hyper-v.txt

for /f %%i in ('findstr /i . hyper-v.txt 2^>nul') do dism /online /norestart /add-package:"%SystemRoot%\servicing\Packages\%%i"

del hyper-v.txt

Dism /online /enable-feature /featurename:Microsoft-Hyper-V-All /LimitAccess /ALL

# 伪装成 win10 专业版

以管理员身份打开 cmd。 执行如下命令：

一定要以管理员身份打开，否则无法执行

REG ADD "HKEY_LOCAL_MACHINE\software\Microsoft\Windows NT\CurrentVersion" /v EditionId /T REG_EXPAND_SZ /d Professional /F
