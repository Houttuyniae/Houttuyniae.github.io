# Hello Word 实例

# 从仓库中下载镜像

docker pull library/hello-world

运行上面的命令，将 image 文件从仓库抓取到本地。

docker pull 是抓取 image 文件的命令。library/hello-world 是 image 文件在仓库里面的位置，其中 library 是 image 文件所在的组，hello-world 是 image 文件的名字。

[root@VM_0_9_centos ~]# docker pull library/hello-world

# 运行镜像

docker run hello-world

会从 image 文件中，创建一个正在运行的容器实例

注意

使用 docker run 命令创建容器时，会先检查本地是否存在指定镜像。如果本地不存在该名称的镜像 ， Docker 就会自动从 Docker Hub 下载镜像并启动一个 Docker 容器。

[root@VM_0_9_centos ~]# docker run hello-world

Hello from Docker!
This message shows that your installation appears to be working correctly.

如果容器没有自动终止，则必须使用 docker run kill [containerID] 命令终止
