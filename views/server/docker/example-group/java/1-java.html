Docker 安装 Java JDK

Houttuyniae 2019-12-20 Docker

# 安装 Nginx

提示

docker 安装 1.8 版本的 Java JDK

# 下载安装

docker pull java:8
1

# 运行 JDK

docker run -it --name java8  -d java:8 /bin/bash
1

# 进入容器

docker exec -it java8 /bin/bash
1

# 验证版本

java - version
1
